About

The Course Site Generator will be a traditional workspace-type application that will let the user create a new course site and export a full generated site as needed.
Debugging Enterprises TM

Site

This tab will allow the user to input the course detail information. Such as course subject, number, semester, year, title, and instructor. This tab also allow you to determine the style of the course site and the pages for it.

Syllabus

This tab allows the user to input information for the syllabus. Such as Topics for the course, textbooks, grading note, etc.

Meeting Time

This tab allows the user to edit the time and dates for lectures, recitations, and labs.

Office Hours

Office Hours allows the user to add office hours for the TAs of the course.

Schedule

Schedule allows the user to add, edit, or remove special events such as holidays, hw due days and lecture cancelled days.
